# 🐾 Animal Image Recognition Website

## 📌 Project Overview
This is a simple AI-based web application that detects animals in an image.  
Users can upload an image, and the system predicts the objects (animals) present in it.

---

## 🚀 Features
- Upload image from browser
- Detect animals using AI model (YOLOv8)
- Display detected objects
- Simple and user-friendly interface

---

## 🛠️ Tech Stack
- Python
- Flask (Backend)
- Ultralytics YOLOv8 (AI Model)
- HTML (Frontend)
