# 🐾 Animal Image Recognition Website

## 📌 Overview
This is a web-based AI application that detects animals in images.  
Users can upload an image, and the system identifies animals using a YOLOv8 model.

---

## 🚀 Features
- Upload image from browser
- Detect animals using AI (YOLOv8)
- Supports multiple object detection
- Displays detected animal names
- Simple and clean UI

---

## 🛠️ Tech Stack
- Python
- Flask (Backend)
- Ultralytics YOLOv8
- Pillow (Image Processing)
- HTML (Frontend)
